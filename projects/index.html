Advanced Algorithm Design Final Project

December 2017

Created in C

For our final project we were given a super interesting situation. Given a maze with two robots we had to find the shortest path to their respective exit points without the two robots coliding. We utilized the Breath Frist Search algorithm and a few cool tricks to complete it. Super fun project I really enjoyed and definitely expanded my thought process for algorithm development.

Minesweeper

December 2016

Created in Javascript utilizing the p5 library

I'm a big fan of minesweeper. With over 1500 games played and a best time 193s for a 100 mine board on the iPhone I would consider myself pretty good. So naturally I decided to make minesweeper as a fun project. I had just learned of the processing library and the newer p5 library based on the original ideas. Most of the functionality is there but it could definitely be better. Maybe when I get some free time I'll come back and take another look at it.
